Dane SUPLA API

Za pomocą poniższego formularza możesz zmienić dane dostępowe do SUPLA API.

Działanie to nie zmieni funkcji SUPLA Scripts, ale jeśli całkowicie zmienisz serwer SUPLA z którym komunikuje się SUPLA Scripts, wysoce prawdopodobne jest że identyfikatory używanych urządzeń lub kanałów zmienią się, co spowoduje że aktualna konfiguracja funkcji dostarczanych przez SUPLA Scripts przestanie działać i będzie musiała być poprawiona.

Ze względów bezpieczeństwa, Twoje dane logowania do SUPLA nie są nigdy przesyłane z serwera do przeglądarki. Jeśli więc nawet chcesz zmienić tylko hasło do API, musisz podać wszystkie poniższe ustawienia jeszcze raz.
